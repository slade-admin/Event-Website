# M&M Events Website

A luxury event planning website for **M&M Events**, built with plain **HTML, CSS, and JavaScript**. The site is designed to present a polished, elegant brand experience for weddings, baby showers, graduations, birthdays, corporate events, and custom celebrations.

## Live pages

- `index.html` — Home
- `about.html` — About
- `services.html` — Services
- `gallery.html` — Gallery
- `testimonials.html` — Testimonials
- `contact.html` — Contact

## Features

- Luxury black-and-gold visual style
- Responsive layout for desktop and mobile
- Static HTML pages with no framework dependency
- Real imagery loaded from local files in `images/`
- Elegant iconography using Font Awesome
- Smooth scrolling and polished section layout
